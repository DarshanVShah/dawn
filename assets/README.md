# Assets for our site

You can add your own photos here and reference them in `index.html` if you like, for example:

- `hero.jpg` — a photo for the top section
- `memory1.jpg`, `memory2.jpg` — for favourite memories
- Any other images you want to use (update the HTML to use `assets/yourfile.jpg`)

The site already uses:
- `pattern-afghan.svg` — background pattern
- `heart.svg` — decorative heart (can be used anywhere you add an img)
